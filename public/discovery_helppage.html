Help

Search

Go to the Find Similar Artists page and type the name of an artist you already like! You will get similar artists.

Tip: Make sure you are spelling the artist's name correctly!

Filter

Use the sliders to narrow the recommendations by a similar artist's monthly listeners and discography play counts. You can filter to find a spectrum of mainstream AND indie artists!

Save

Press the heart button next to an artist to add them to your Favorite Artists page.

Navigation Tip

Use the headphone icon in the top left corner to return to the home screen, or use the bottom navigation bar to switch pages.
